# CareHub - Your Health, Our Priority

Welcome to **CareHub**, a modern, full-stack healthcare platform built with Next.js and Firebase. This application provides a seamless experience for both patients and doctors, offering services like appointment booking, AI-powered symptom checking, and secure messaging.

## About The Project

CareHub is designed to bridge the gap between patients and healthcare providers through an intuitive and feature-rich web application. It demonstrates the power of modern web technologies in creating accessible and efficient healthcare solutions.

### Key Features

-   **Patient & Doctor Dashboards**: Separate, tailored experiences for patients and doctors.
-   **Appointment Booking**: Easily find doctors and book appointments based on their availability.
-   **AI Symptom Checker**: An intelligent tool to help patients identify potential specialties to consult.
-   **Secure Chat**: Real-time, private communication between patients and doctors.
-   **Medical Records Management**: Patients can view their medical history and records.
-   **Online Pharmacy**: A browsable list of medications.

## Built With

This project is built on a modern, robust tech stack:

-   [Next.js](https://nextjs.org/) - React framework for production
-   [React](https://reactjs.org/) - A JavaScript library for building user interfaces
-   [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript at scale
-   [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework
-   [ShadCN UI](https://ui.shadcn.com/) - Re-usable components built using Radix UI and Tailwind CSS
-   [Genkit (for AI)](https://firebase.google.com/docs/genkit) - A framework for building AI-powered features
-   [Firebase](https://firebase.google.com/) - For authentication and backend services

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need `npm` installed on your machine.

### Installation & Running the App

1.  **Install dependencies**:
    ```sh
    npm install
    ```
2.  **Run the development server**:
    ```sh
    npm run dev
    ```
3.  Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

The main page of the application can be found at `src/app/page.tsx`. You can start by editing this file.
