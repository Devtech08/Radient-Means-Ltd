# Radiant Means Ltd. - Corporate Services Platform

This is a Next.js web application for Radiant Means Ltd., a diversified services company.

The application serves as the corporate face for Radiant Means Ltd., showcasing the various services the company offers.

## Key Features

- **Home Page:** A modern landing page introducing Radiant Means Ltd. and its motto, "Service based on Due Diligence."
- **About Us:** Information about the company's mission, goals, and the core values that drive its commitment to excellence.
- **Services:** A detailed look at the various services offered by the company, including Transportation, General Trading, Civil Engineering, Cleaning Services, Electrical Engineering, Real Estate Development, and Security Services.
- **Why Choose Us:** Highlighting the company's strengths, extensive experience, and commitment to quality.
- **Contact Us:** A functional contact form and company contact details for inquiries.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [ShadCN UI](https://ui.shadcn.com/)
- **AI/Generative:** [Google's Genkit](https://firebase.google.com/docs/genkit) (for the contact form action)
- **Forms:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)

## Getting Started

Follow these steps to get the project running locally.

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file in the root of the project if you need to add any specific environment variables. For this project, core services do not require additional environment variables to run locally.

### Running the Development Server

To start the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
